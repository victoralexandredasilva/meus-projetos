HTML5 + CSS3

Projeto Cordel

O projeto cordel foi feito por mim através do curso de HTML5 e CSS3 do CursoemVídeo com o Professor Gustavo Guanabara.
O projeto foi feito com responsividade, ou seja ele foi feito para que sejá bem receptivo com diversos tipos de telas, sendo das telas pequenas, como celulares, até para as telas grandes, como desktops até TV's.
O site foi feito com boas técnicas, um deles por exemplo é o Parallax, que é uma das tecnicas que dá ao site uma aparência muito boas, deixando a imagem de fundo fixa enquanto contéudo segue rolando.
